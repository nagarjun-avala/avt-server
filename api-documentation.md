# API Documentation - Inventory Management System (IMS)

## Authentication API

### POST /api/login

**Request**:

```json
{
  "username": "admin",
  "password": "password123"
}
```

**Response**:

```json
{
  "token": "jwt-token",
  "user": {
    "data": "User details"
  }
}
```
